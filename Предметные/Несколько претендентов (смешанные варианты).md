### Если на одну свободную комнату претендуют двое:
## 1 - по льготной стоимости, 2 - по рыночной стоимости:
Приоритетное право покупки у заявителя № 2( по рыночной стоимости). Гражданину № 1 предлагается также выкупить по рыночной стоимости.
## 1 - по льготной стоимости, 2 - по льготной стоимости:
В выкупе участвуют оба, право собственности на выкупаемое жилое помещение предоставляется в долях пропорционально имеющимся долям в праве собственности на жилую площадь в занимаемых комнатах.
## 1 - по рыночной стоимости, 2 - бесплатно по ДСН:
В данном случае свободное жилое помещение будет оформлено в равнодолевую собственность претендующих
## 1 - по рыночной стоимости, 2 - по рыночной стоимости:
В данном случае свободное жилое помещение будет оформлено в долях пропорционально имеющимся долям в праве собственности на жилую площадь в занимаемых комнатах.
## 1 - по рыночной стоимости ( с регистрацией по месту жительства), 2 - по рыночной стоимости (без регистрации по месту жительства:
Продажа возможна только гражданину № 1 (с регистрацией по месту жительства).

*В случае, если один из претендентов оплатил свою часть, а второй - нет, то отказ обоим.
